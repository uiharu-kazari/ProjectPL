Qu'Est-ce que c'est?

  • Financial Data have internal structures
  • Expressed as points in the Euclidean space Rn\mathbb{R}^nRn, these point cloud will have a shape
  • We believe that shape has meaning,
  • And meaning drives value.
  • Topological Data Analysis (TDA) is a set of tools used to extract information about the shape

Our Technology

  • We can reproduce the result in the paper Topological data analysis of financial time series: Landscapes of crashes
  • This paper tried to devise new early warning signals (EWS) to predict incoming financial crashes.
  • Spoiler: It is a nice try, but at least it provides an innovative tool for analysis.

Hypothesis:

correlation breakdowns

  • We assume that during (or prior to?) market crashes, assets that were correlated would become less correlated (See this paper).
  • To a certain extent, this correlation breakdown can be measured by the appearance of loops in high dimensional spaces. (If they were perfectly (linearly) correlated, we would observe a straight line.)
  • We use tools from TDA to give a quantified value to the appearance of loops.

Pipeline

  • Data: DJIA, NASDAQ, RUSSELL 2000, SP500
  • For each trading day, we put the adjusted closing price of them in an array, this gives us a point in R4\mathbb{R}^4R4 for each day.
  • Select a size for the sliding window, say 50 days. Then for day T, we formed a point cloud consists of points from day T-49 to day T, call this point cloud PC(T).
  • Then we compute the so-called p-norm of the persistence landscape associated with PC(T)
  • Do this for all available trading days, we obtain a time series, and can compare this with the index of SP500, for instance.

Observation and Comments

  • The appearance of very high 1-norms are all results of plunges already happed. So technically, they are not EWS of future crashes.
  • This contradicts the claim in the previously referred paper.
  • See a calculation in the persistence landscape of dimension 2
  • Future work: we may include more asset types, which would possibly equip us better ability to detect correlation breakdowns.
  • Note: during crashes, assets not correlated before can become correlated as well. We may incorporate that insight in future strategies.
